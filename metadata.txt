version=1.3.0
author=Dorset Council
email=gis@dorsetcouncil.gov.uk

about=This plugin adds a configurable search panel to QGIS as a dockable widget. Users can search through multiple APIs and project layers with customizable templates, ordering, and regex filtering. Features include: interactive results list with clickable items, API endpoint configuration with URL templates, QGIS Authentication Configuration support for secure API access, search priority ordering, regex-based relevance filtering, project layer integration, visual search feedback, persistent results display, loading indicators, smart zoom levels, dockable panel interface, and configurable search stopping behavior.
